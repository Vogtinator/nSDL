So what's this nSDL thing about?

nSDL is a port of SDL for the TI-Nspire calculator. If you're unsure what SDL is, here's a short summary from the official website:

Simple DirectMedia Layer is a cross-platform multimedia library designed to provide low level access to audio, keyboard, mouse, joystick, 3D hardware via OpenGL, and 2D video framebuffer. It is used by MPEG playback software, emulators, and many popular games, including the award winning Linux port of "Civilization: Call To Power."

In much more general terms, nSDL makes game development on the TI-Nspire a lot easier (although it is not restricted for games only). It also allows computer programs that use SDL (some examples here and here) to be ported to the said calculator.

Its features include:

  • Full one-binary-for-all support on every TI-Nspire (CAS & non-CAS) calculator—Clickpad, Touchpad & CX.
  • Very fast & stable buffered drawing.
  • Supports all graphics-related functions, events, non-threaded timer functions, and more.
  • Supports 8- and 16-bit display bit depth.
  • Based on the robust SDL 1.2.13.

Alrighty then, what now?

If you're new to native TI-Nspire programming and/or want to know how to easily set up your Ndless development environment and install nSDL, move on to the how-to guide. You can view a concise list of functions available in SDL here. You can access code examples and other interesting stuff through the SDL documentation's front page. There might also be some useful pieces of information on the nSDL wiki, so make sure to check it out.

Download & source code

If you're looking for the files necessary to make your own SDL programs on the TI-Nspire, just click on the big blue button up there—you can't miss it. The attractive-looking folders at the right are only for those interested in the latest source code, or for those who want to build nSDL from scratch.
